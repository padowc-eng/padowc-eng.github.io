Purpose of this page

Not every design project needs its own full page. This page is for smaller but still useful examples: mechanisms, fixtures, CAD assemblies, class projects, prototype parts, FEA-supported parts, or personal design work.

Entry format

Each smaller project should follow the same simple structure so the page stays readable. 

// small project entry structure
project_name: "short name"
problem: "what needed to be designed or solved"
work_done: "CAD, calculations, prototype, analysis, drawing, etc."
result: "what changed, improved, or was learned"
